## Portfolio Structure

**Part 1 — Prompt Design & Iteration**  
Three content generation scenarios — technical explanation, professional email, and creative marketing content — each refined through multiple iterations with documented improvements. Includes role assignment and context setting analysis.

**Part 2 — Temperature & Parameter Control**  
Experimentation with low, medium, and high temperature settings on a photosynthesis explanation task, with a comparison table and analysis of when each setting is most appropriate.

**Part 3 — Strategic Prompting Techniques**  
Demonstrates chain-of-thought prompting on a multi-step planning problem and few-shot prompting for sentiment classification, with before-and-after comparisons for both techniques.

**Part 4 — Responsible AI & Limitations**  
Critical examination of LLM hallucinations using a real-world fashion event test case, cultural bias testing on professional attire recommendations, and a reflection on limitations and responsible use practices.
